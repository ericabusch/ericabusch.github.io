  • Representations for Biomarkers

    Repository for reproducible biomarker discovery in neuroimaging, with scripts, workflows, and tutorials for functional alignment and manifold learning approaches. Includes Google Colab notebook tutorials for working with provided simulated and sample fMRI data. Accompanies Busch et al., 2025 (in revision).

  • Exogenous-PHATE analyses

    Analysis code and E-PHATE method; combining adolescent brain and environment features to predicting psychopathology development, as published in Biological Psychiatry: Cognitive Neuroscience and Neuroimaging (2024).

  • Temporal-PHATE analyses

    Analysis code and tools for the T-PHATE manifold learning method for fMRI timeseries, as described in "Multi-view manifold learning of human brain-state trajectories" (Nature Computational Science, 2023).

  • avatarRT Unity Task

    Unity-based navigation game for fMRI, designed for the avatarRT project. Provides a customizable environment and video game for use with closed-loop experiments, designed in Unity3D (C#).

  • Neurofeedback avatarRT Analyses

    Analysis pipeline for AvatarRT real-time fMRI neurofeedback experiment, including rt-cloud interface, data preprocessing, neural and behavioral analyses, statistis, and visualization tools, to go with Busch et al., 2025 (biorxiv)

  • ABCD - Hyperalignment

    Analysis scripts and CHA implementation for measuring reliability, heritability, and predictivity of developing functional connectomes, as published in Journal of Neuroscience (2024).

  • Hybrid Hyperalignment (H2A)

    Code for hybrid hyperalignment: a high-dimensional model of shared information in cortical patterns of response and functional connectivity, as published in NeuroImage (2021).
